# Taichi 粒子引力特效（Experiment 0: Taichi Gravity Swarm）
这是一个基于 Taichi 图形库实现的简单粒子引力交互特效，通过鼠标移动控制粒子的引力吸引，粒子会跟随鼠标运动并在窗口边界反弹，具备空气阻力物理效果。

## 项目介绍
### 功能特点
- 基于 GPU 并行计算的高性能粒子渲染（支持万级粒子数量）；
- 鼠标实时引力交互：粒子会向鼠标位置聚集，移动鼠标可牵引粒子群；
- 物理效果模拟：包含空气阻力、窗口边界反弹能量损耗；
- 跨平台运行：支持 Windows/macOS/Linux，依赖 Taichi 图形库。

### 技术栈
- 编程语言：Python 3.8+
- 核心库：Taichi（版本 ≥ 1.7.0）

## 快速开始
### 环境准备
1. 安装 Python 环境（推荐 3.8-3.12 版本）；
2. 安装 Taichi 库：
   ```bash
   pip install taichi>=1.7.0
   ```

### 运行项目
1. 克隆/下载本项目代码到本地；
2. 进入项目根目录，执行主文件：
   ```bash
   python main.py
   ```
3. 程序会先编译 GPU 内核，编译完成后弹出交互窗口，移动鼠标即可控制粒子引力。

## 代码结构说明
```
├── main.py          # 项目主入口（核心运行逻辑）
└── src/
    └── Work0/
        ├── config.py # 系统参数配置（粒子数量、物理系数、渲染参数）
        ├── physics.py # 物理核心逻辑（粒子初始化、引力计算、碰撞检测）
        └── main.py   # 子模块运行入口（与根目录main.py功能一致）
```

### 核心文件说明
| 文件 | 作用 |
|------|------|
| `config.py` | 定义所有可配置参数（粒子数量、引力强度、阻力系数等） |
| `physics.py` | 实现粒子初始化、GPU 并行物理计算（引力、阻力、边界反弹） |
| `main.py`（根目录） | 项目主运行入口，初始化 Taichi、创建 GUI 窗口、渲染主循环 |

## 配置参数说明（config.py）
可根据需求调整以下参数，优化运行效果：
| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `NUM_PARTICLES` | 10000 | 粒子总数（卡顿可调小，如 2000/5000） |
| `GRAVITY_STRENGTH` | 0.001 | 鼠标引力强度（值越大，粒子被吸引得越快） |
| `DRAG_COEF` | 0.98 | 空气阻力系数（值越小，粒子减速越快） |
| `BOUNCE_COEF` | -0.8 | 边界反弹能量损耗（绝对值越小，反弹后速度越慢） |
| `WINDOW_RES` | (800, 600) | 窗口分辨率（宽，高） |
| `PARTICLE_RADIUS` | 1.5 | 粒子绘制半径（值越大，粒子显示越大） |
| `PARTICLE_COLOR` | 0x00BFFF | 粒子颜色（16 进制 RGB，0x00BFFF 为天蓝色） |

## 运行效果
- 程序启动后，粒子随机分布在窗口内；
- 移动鼠标，粒子会向鼠标位置持续聚集；
- 粒子碰到窗口边缘会反弹，并损失部分动能；
- 停止移动鼠标后，粒子因空气阻力逐渐减速，最终停在鼠标位置附近。

## 常见问题
1. **运行卡顿**：
   - 降低 `NUM_PARTICLES` 参数（如改为 2000）；
   - 若显卡不支持 GPU，修改 `ti.init(arch=ti.gpu)` 为 `ti.init(arch=ti.cpu)`。
2. **编译失败**：
   - 确保 Taichi 版本 ≥ 1.7.0，执行 `pip install --upgrade taichi` 升级；
   - 检查 Python 版本是否为 3.8+。
3. **窗口无响应**：
   - 关闭其他占用 GPU/CPU 的程序，重新运行；
   - 减少粒子数量后重试。

## 操作说明
- 移动鼠标：控制粒子引力中心，粒子会跟随鼠标运动；
- 关闭窗口：点击窗口右上角关闭按钮，或按 `ESC` 键。
