2.  独立推导并代码实现**模型变换（Model）、视图变换（View）、透视投影变换（Projection）** 三个核心4×4齐次坐标变换矩阵。
3.  掌握Taichi面向数据编程框架的基本语法、矩阵操作与GPU/CPU并行计算逻辑。
4.  （选做）实现3D立方体的透视旋转渲染，验证3D空间变换的立体效果，深入理解透视投影的近大远小特性；实现两个不同姿态立方体间的旋转插值平滑过渡动画。

## 五、核心实现内容
### （一）基础任务：三角形MVP变换
#### 1. 核心矩阵实现
严格按照图形学标准管线，实现三个核心变换矩阵：
1.  **模型变换矩阵 `get_model_matrix`**
    - 功能：接收角度制旋转角，返回绕Z轴的4×4旋转矩阵
    - 实现：先完成角度→弧度转换，再构建标准的Z轴旋转矩阵
2.  **视图变换矩阵 `get_view_matrix`**
    - 功能：接收相机三维位置，返回视图变换矩阵
    - 实现：将相机平移至世界坐标系原点，构建平移逆矩阵
3.  **透视投影矩阵 `get_projection_matrix`**
    - 功能：接收视场角、宽高比、近/远截面距离，返回透视投影矩阵
    - 实现：分两步完成——先将透视平截头体挤压为正交长方体，再通过正交投影矩阵归一化到[-1,1]的NDC标准设备坐标系

#### 2. 渲染管线流程
1.  对三角形的3个顶点扩展为4维齐次坐标
2.  按列向量右乘规则，组合MVP矩阵：`MVP = Projection @ View @ Model`
3.  应用MVP矩阵完成顶点的空间变换
4.  执行透视除法，将齐次坐标归一化到NDC坐标系
5.  视口变换，将NDC坐标映射到GUI屏幕坐标，完成线框绘制

### （二）选做任务：3D立方体透视旋转与插值动画
1.  **3D几何体构建**：定义中心在原点、边长为2的立方体，包含8个顶点（坐标范围[-1,1]）、12条边的线框索引，适配Taichi语法规范避免索引报错
2.  **完整MVP渲染管线**：复用基础任务的图形学标准管线，实现绕X轴+Y轴的双轴旋转，通过透视投影矩阵实现近大远小的立体视觉效果
3.  **旋转插值平滑过渡**：实现两个自定义旋转姿态间的线性插值逻辑，通过速度反向实现0→1→0的平滑往返动画，无生硬跳变
4.  **渲染逻辑升级**：遍历立方体的12条边完成线框绘制，适配Taichi 1.7.4版本语法规范，保证代码稳定运行
