# 计算机图形学实验三：贝塞尔曲线实现
**课程**：北京师范大学人工智能学院 计算机图形学（2026春）  
**授课教师**：张鸿文  
**实验完成时间**：2026年4月  
**技术栈**：Python 3.12 + Taichi 1.7.4

## 实验概述
本实验实现了基于De Casteljau算法的贝塞尔曲线绘制，并完成了反走样、均匀三次B样条曲线、控制点拖拽、撤销等选做功能。通过CPU批量计算+GPU并行渲染的架构，实现了高帧率的实时交互效果。

## 文件说明
| 文件名 | 功能 |
|--------|------|
| `lab3base.py` | 基础版：仅实现实验要求的必做功能 |
| `lab3bonus.py` | 完整版：包含所有必做+选做功能 |

## 功能实现
### ✅ 必做功能（全部完成）
- 鼠标左键点击添加红色控制点
- 实时绘制灰色控制多边形
- De Casteljau递归线性插值算法生成贝塞尔曲线
- 800x800像素缓冲区手动光栅化
- GPU批量并行绘制（1000个采样点，帧率稳定300+）
- C键一键清空画布
- 预分配显存对象池优化，避免动态内存申请

### ✅ 选做功能（全部完成）
1. **反走样平滑曲线**
   - 基于3x3邻域距离衰减模型
   - 高斯权重混合，消除曲线锯齿边缘
   - A键一键开关，可直观对比效果

2. **均匀三次B样条曲线**
   - 矩阵形式实现，避免复杂递归
   - 分段平滑拼接，n个控制点生成n-3段曲线
   - B键一键切换贝塞尔/B样条模式
   - 支持局部控制：移动中间控制点仅影响附近曲线段

3. **增强交互**
   - 按住控制点拖拽，曲线实时更新
   - Backspace键撤销上一个添加的控制点

## 操作说明
| 操作 | 功能 |
|------|------|
| 鼠标左键点击空白处 | 添加新控制点 |
| 鼠标左键按住红色圆点拖拽 | 移动控制点 |
| Backspace 键 | 撤销最后一个控制点 |
| C 键 | 清空所有内容 |
| B 键 | 切换 贝塞尔曲线 / 均匀三次B样条曲线 |
| A 键 | 开启/关闭反走样效果 |
